# SoloTravel/SoloSafe Front-end style Guide

## Layout
Design Principles
Safety First
Every decision prioritizes user safety, with clear visual hiearchy for emergency features and critical actions.
Trust & Transparency
Clean, professional aesthtics that convey reliability. No dark patterns., clear
comuunication of data usage.

Accesibility
High contrast ratios(WCAG AA compliant), generous touch targets (44px minimum), and clear visual feedback.

Responsive & Mobile-first
Optimized for on-the-go usage with mobile-first design approach and touch-friendly interactions.
